# Snowflake Validation Checklist (Demo)

Use this checklist to validate the demo against a target Snowflake account/version.

## Account and Role
- `SHOW ROLES LIKE 'TELCO_%';`
- `SHOW USERS LIKE 'TELCO_MCP_USER';`
- `SHOW WAREHOUSES LIKE 'TELCO_ASSURANCE_WH';`

## Database, Schema, and MCP Server
- `SHOW DATABASES LIKE 'TELCO_AI_DB';`
- `SHOW SCHEMAS IN DATABASE TELCO_AI_DB;`
- `SHOW MCP SERVERS IN SCHEMA TELCO_AI_DB.NETWORK_ASSURANCE;`
- `DESCRIBE MCP SERVER TELCO_AI_DB.NETWORK_ASSURANCE.TELCO_ASSURANCE_MCP;`

## Grants
- `SHOW GRANTS TO ROLE TELCO_SN_INTEGRATION_RL;`
- `SHOW GRANTS ON MCP SERVER TELCO_AI_DB.NETWORK_ASSURANCE.TELCO_ASSURANCE_MCP;`
- `SHOW GRANTS ON ALL TABLES IN SCHEMA TELCO_AI_DB.NETWORK_ASSURANCE;`

## Data Load Verification
- `SELECT COUNT(*) FROM NETWORK_KPI;`
- `SELECT COUNT(*) FROM ALARMS;`
- `SELECT COUNT(*) FROM INCIDENTS;`
- `SELECT COUNT(*) FROM TROUBLE_TICKETS;`
- `SELECT COUNT(*) FROM CMDB_CI;`

## MCP Connectivity
- `tools/list` from your MCP client.
- `tools/call` with a simple query, e.g. `SELECT COUNT(*) FROM NETWORK_KPI;`
